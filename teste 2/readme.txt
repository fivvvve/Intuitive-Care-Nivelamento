To run the code do the following:

1. Make sure you have java installed and configured in your environment

2. Create and access a virtual enviremont using:

Ubuntu:
python3 -m venv .venv
source .venv/bin/activate

Windows:
python -m venv .venv
.venv\Scripts\activate

3. Install the used packages:
pip install -r requirements.txt

4. Run the code:
python main.py
